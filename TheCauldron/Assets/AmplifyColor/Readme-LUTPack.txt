Description

  Over 200 LUTs in a comprehensive and varied pack exclusively for Amplify Color, ready
  to use and extremely simple to add to your project. Blend up to 3 LUT’s with the added
  blending features available in Amplify Color along with Volume and Depth based Masking
  and easily apply new color profiles using the new LUT Browser. 
	
In the package

Minimum Requirements

LUT Browser Quick Guide
  
  1) Open & Dock LUT Browser.
  2) Add the Amplify Color Component to your active camera or use an existing amplified camera.
  3) Select the preview camera in the LUT browser window and left click on a thumbnail to apply it.   
  4) To use a preview image deselect the preview camera and select a thumbnail.
  5) Left Click selects a LUT and Right Click will locate the LUT in your project folder; LUTs can 
     be dragged and dropped to the Amplify Color Component or Amplify Color Volumes.
